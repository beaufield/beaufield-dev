# オートシャンプー展示会 予約システム

Google Apps Script（GAS）で動作するWebアプリです。
サーバー不要・無料・Googleアカウントだけで完結します。

---

## ファイル構成

```
このフォルダ/
├── Code.gs       → GASにコピーするバックエンドコード
├── index.html    → GASにコピーする予約ページ（来場者向け）
├── admin.html    → GASにコピーする管理画面（社内向け）
└── README.md     → この手順書
```

---

## セットアップ手順（約20分）

### ① Googleスプレッドシートを作成する

1. Googleドライブで「新規」→「Googleスプレッドシート」を作成
2. 名前を「展示会予約データ」などに変更
3. URLの中の長い英数字をコピーしておく
   - 例：`https://docs.google.com/spreadsheets/d/`**ここがID**`/edit`

---

### ② Google Apps Scriptプロジェクトを作成する

1. Googleドライブで「新規」→「その他」→「Google Apps Script」
2. プロジェクト名を「展示会予約システム」などに変更

---

### ③ コードをコピーする

#### Code.gs
1. GASエディタの左側「コード.gs」をクリック
2. 中身を全部削除して、`Code.gs` の内容をまるごとペースト
3. 上部の設定3箇所を編集する：
   ```javascript
   const SPREADSHEET_ID    = 'ここにスプレッドシートのIDを貼る';
   const LINEWORKS_WEBHOOK = 'ここにLINE WORKS WebhookのURLを貼る';
   const ADMIN_PASSWORD    = 'beaufield2026'; // ← お好みで変更
   ```

#### index.html（HTMLファイル）
1. GASエディタ左側「＋」→「HTMLファイル」をクリック
2. ファイル名に `index` と入力（.htmlは自動でつく）
3. 中身を全部削除して、`index.html` の内容をまるごとペースト

#### admin.html（HTMLファイル）
1. 同様に「＋」→「HTMLファイル」で `admin` を作成
2. `admin.html` の内容をまるごとペースト

---

### ④ スプレッドシートを初期設定する

1. GASエディタで関数選択リストから `setupSheet` を選ぶ
2. 「実行」ボタンを押す
3. 初回は「権限を確認」→「詳細」→「許可」でOK
4. スプレッドシートにヘッダー行が作成されれば完了

---

### ⑤ Webアプリとしてデプロイする

1. GASエディタ右上「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」
3. 設定：
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: 全員
4. 「デプロイ」→「アクセスを承認」→「許可」
5. 表示された **ウェブアプリのURL** をコピー

---

### ⑥ LINE WORKS Webhookを設定する

1. LINE WORKS 管理者コンソールを開く
2.「連携サービス」→「Incoming Webhook」→「追加」
3. Bot名「展示会予約通知」などで作成
4. 表示された Webhook URL を Code.gs の `LINEWORKS_WEBHOOK` に設定
5. GASで再デプロイ（「デプロイ」→「デプロイを管理」→「編集」→バージョンを「新バージョン」→「デプロイ」）

---

## URL一覧

デプロイ後のURLを使います。

| 用途 | URL |
|------|-----|
| 予約ページ（来場者に送る） | `https://script.google.com/macros/s/〇〇〇/exec` |
| 管理画面（社内用） | `https://script.google.com/macros/s/〇〇〇/exec?page=admin` |

---

## 管理画面のパスワード

デフォルト：`beaufield2026`

Code.gs の `ADMIN_PASSWORD` を変更して再デプロイすれば変わります。

---

## コードを変更したときの再デプロイ手順

1. GASエディタで「デプロイ」→「デプロイを管理」
2. 鉛筆アイコン（編集）をクリック
3. バージョンを「**新バージョン**」に変更
4. 「デプロイ」ボタンを押す
5. URLは変わらないのでそのまま使えます

## 仕様メモ

- **期間**: 2026年4月1日（水）〜 4月7日（火）
- **時間帯**: 9:00〜16:30（30分刻み、計16枠/日、112枠/全体）
- **複数枠**: 1回の予約で複数枠を選択可能
- **重複防止**: 同一枠への同時予約は自動でブロック（LockServiceを使用）
- **予約台帳**: Googleスプレッドシートに自動保存
- **社内通知**: 予約完了時にLINE WORKSへ自動通知

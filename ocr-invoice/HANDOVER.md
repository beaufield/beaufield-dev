## 重要な技術メモ

### 商品マスターCSVの文字コード
- 販売管理システム出力は **CP932**（Windows拡張Shift-JIS）
- `shift_jis` ではなく **`cp932`** で読む（convert.py対応済み）
- **ExcelでCSV保存するとJANが科学表記（`4.5266E+12`）に壊れる**
  → 必ずconvert.pyで生成すること。Excel保存は絶対NG

### CSV文字化け対策
- Claude Codeが出力するCSVはUTF-8（BOMなし）
- ExcelはBOMなしUTF-8を文字化けして開く
- **add_bom.bat を実行してからExcelで開くこと**
- 将来的にはスキャン処理に組み込む予定

### BATファイルの注意点
- bat/ps1をUTF-8で保存すると日本語パスが化けて失敗する
- **解決策：bat内の日本語を排除し、Pythonに委譲する構成**
  （csv変換.bat → convert.py）

### JANコード照合の注意点
- 商品コード列（相手商品CD）はメーカーごとに形式が異なる
- シュワルツコフは「数字のみ（例：2962558）」＝ 相手商品CDの数字部分
- 日華化学は「S+数字（例：S39）」
- JANコード13桁が最も信頼性が高い照合キー

### OCRの読み誤り実績（シュワルツコフ）
前回セッションでOCRが以下を誤読した。次回も同様のパターンに注意：
- 「パウダーブリーチ」→「カラーアルカレ」と誤読
- 「クレイブリーチ」→「クラフトリーナ」と誤読
- 製品コードを隣の行のものと混同することがある
→ **未照合が出たらPDFを再読みして製品コード・JANを直接確認すること**

## 照合ロジック

| 優先度 | 方法 | 信頼度 |
|---|---|---|
| 1 | JANコード完全一致（13桁） | 🟢 高 |
| 2 | 相手商品CD完全一致（S39, C01等） | 🟢 高 |
| 3 | 商品名部分一致（最大3候補） | 🟡 中 |
| 4 | 未照合 | 🔴 要手動対応 |

## 出力CSVの形式

**ファイル名**：`{メーカー名}_{納品日YYYYMMDD}.csv`

**内容（照合済み）**：
```
コード,商品名,数量,単価,合計
7851,FIBREPLEXファイバープレックス パウダーブリーチ 450g,24,2850,68400
...
```

**未照合がある場合**（末尾に追記）：
```
【未照合商品】
OCR商品名,OCR製品コード,数量,単価,合計
...
```
